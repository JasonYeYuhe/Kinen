Hello App Review,

Yes, the app includes optional WeatherKit functionality.

How to access it on macOS:
1. Launch Kinen.
2. Open Settings from the sidebar.
3. In the "Journal" section, enable "Auto-record location & weather".
4. Grant location permission when macOS prompts for it.
5. Return to the Journal screen.
6. Click the "New Entry" button in the toolbar (square-and-pencil icon).
7. When the new entry editor opens, the app fetches the current location and weather in the background.
8. Save the entry. The saved entry will display the location and weather metadata in the entry detail view.

Additional context:
- This feature is optional and is off by default.
- Weather data is only fetched for new entries after the user enables the setting above.
